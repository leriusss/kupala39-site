# КупалаФЕСТ / kupalafest39.ru

## Роль проекта

Это промо-сайт и рабочий пакет материалов для фестиваля Ивана Купалы в Птичьем Дворе.

Пользователь согласовывает решения. Ассистент ведет проектную логику, структуру задач, тексты, сайт и материалы для соцсетей.

## Рабочие принципы

- Используем реальные фото и видео, не генерируем изображения фестиваля.
- Не выдумываем факты: билеты, парковку, программу, контакты и погодный план помечаем как требующие подтверждения, пока организаторы не дали точные данные.
- Лендинг должен быть простым, быстрым и мобильным.
- Визуальный стиль: зелень, молочный/белый, теплый желтый, дерево, сено, мягкий оранжевый/терракотовый, вечерний свет.
- Избегаем перегруженных афиш и длинных текстов на визуалах.

## Текущая структура

- `index.html` — статичный лендинг.
- `social-pack.md` — тексты для VK, Telegram, партнерских посевов, блогеров, Stories/Reels.
- `README.md` — краткая инструкция по запуску и DNS.

## Ближайшие задачи

1. Получить реальные фото/видео и заменить визуальные заглушки.
2. Подтвердить программу, стоимость входа, парковку, контакты и погодный план.

4. Подготовить OG-картинку для соцсетей.
5. Опубликовать сайт на `kupalafest39.ru`.
